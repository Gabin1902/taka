Read the Docs
=============

HTML Format
-----------

    open build/html/index.html

PDF Format
----------

    open build/latex/lambdaconcepttaka.pdf

Building the documentation
==========================

Dependencies
------------

    python-sphinx
    (for pdf) texlive

Generate as html
----------------

    make html
    open build/html/index.html

Generate as pdf
---------------

    make latexpdf
    open build/latex/lambdaconcepttaka.pdf
